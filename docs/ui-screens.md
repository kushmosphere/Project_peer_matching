# UI Screens

## Visual Direction

- Dark mode first with high-contrast panels.
- Instagram-inspired bottom navigation and scrolling media feed.
- Bumble-style full-height swipe cards with clear mode toggle.
- 8px radius for controls and restrained rounded corners for phone-native cards.
- Gen-Z energy through color accents, motion, microinteractions, and expressive profile media.
